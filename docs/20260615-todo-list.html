TradingAssistant Frontend TODO List

生成日期：2026-06-15  |  基于 itick SDK v0.1.0 + OpenTrade v1.1.0  |  当前前端状态：100% Mock

核心结论：itick SDK 是纯粹的行情数据供应商——它通过 REST + WebSocket 提供股票/债券/基金/外汇/加密货币的 tick（逐笔）、quote（报价）、depth（深度）、kline（K线）裸数据。

本项目的 frontend 是一套完整的多资产交易终端分析系统，在 itick 提供的裸数据之上构建了 技术指标计算、多周期聚合、投研分析框架、AI 市场解读、专业图表渲染、跨资产宏观仪表板和终端级交互体验共 7 个增值层。

以下将所有功能拆分为「itick 原生支持」和「需自行实现」两大类，每项附当前完成状态。

✅ 第一部分：itick SDK 原生支持（无需额外实现）

以下能力直接调用 itick SDK 即可获得，不需自行开发数据获取逻辑。

# 功能 itick 对应 API 状态 说明
1 股票实时报价 get_stock_quote / get_stock_quotes itick 原生 返回最新价、开盘、最高、最低、昨收、成交量
2 股票历史 K 线 get_stock_kline / get_stock_klines itick 原生 返回 OHLCV 结构，支持 period / limit / end 参数
3 市场深度数据 get_stock_depth / get_stock_depths itick 原生 返回买卖盘口档位数据
4 逐笔成交数据 get_stock_tick / get_stock_ticks itick 原生 逐笔价格、成交量、方向
5 实时 WebSocket 推送 connect_stock_websocket + subscribe itick 原生 支持 tick/quote/kline/depth 四类 topic 订阅

注意：以上 5 项仅指"数据获取"环节。数据拿到后仍需经过 normalizer → 事件模型转换 → 聚合 → 指标计算 → 渲染等链路才能真正呈现在 frontend 上。 这些链路中的大部分步骤属于"需自行实现"范畴。

⚠️ 第二部分：通过组合 itick 查询可获取（需明确代码映射）

功能本身在 itick API 能力范围内，但需要跨资产类别查询或配置具体代码映射。

# 功能 itick 对应 API 状态 说明
1 Top Bar 宏观指标条 (UST10Y / DXY) get_bond_quote + get_forex_quote 需组合 需确定各指标对应的 region+code；当前为硬编码 mock 值
2 商品期货 (WTI / Gold) itick 无商品期货端点 需额外数据源 itick 支持股票/债券/基金/外汇/加密货币，不含商品期货
3 VIX 恐慌指数 需要美股指数衍生品数据 需额外数据源 VIX 是 CBOE 波动率指数，itick 未明确支持

建议：UST10Y/DXY 可通过 itick 获取；WTI/Gold/VIX 需要评估是否引入额外数据源（如 yfinance、Alpha Vantage） 或暂时保持 mock。
